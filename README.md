# プロジェクトの説明

・このプロジェクトはNotionに保存されている中国語のサイトを翻訳して、新たにページを起こしてくれるサービスです。
・中国語のサイトには多数の画像が使われていますが、画像の中の文字を読めるように文字を一対一で読めるように別表を付けて日本人でも理解してくれるようにします。しかし、この表をベタうちで貼り付けると非常に読みにくいのでtoggle形式で閉じれるようにしておきます。
・これらのページは、基本的に全世界にオープンにして見たい人がいつでも見られるようにしておきます。
・日本語に翻訳した中国語のサイトは再度、翻訳にかけられないように管理する必要があります。翻訳が完了したら、titleの冒頭にdoneと文字を書くようにしてください。

# プロジェクト開発にあたっての前提条件
・configフォルダに関連する中国語サイトの情報を取得するためのIDや展開先の日本語サイトのID，Notionにアクセスするためのapiの記載があります。
・ChatGPTのAPIは
import os
openai.api_key = os.getenv("OPENAI_API_KEY")
で取得できます。
・configフォルダに設計したpromptを使って、翻訳を実行してください。
